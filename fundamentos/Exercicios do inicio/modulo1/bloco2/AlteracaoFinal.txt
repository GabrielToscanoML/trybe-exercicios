5 motivos para você aprender JavaScript!
------------------------------------------

O JavaScript é uma linguagem de programação muito poderosa utilizada em todo o mundo!

O uso mais convencional do JavaScript é no contexto do nosso navegador, juntamente com HTML e CSS.

Por que o JavaScript é tão valorizado?
----------------------------------------

O JavaScript te permite dar dinamismos às páginas. É ele quem garante que sua página tenha a habilidade de reagir às ações das pessoas usuárias.

5 motivos para você aprender JavaScript
-----------------------------------------

#1 Podendo ser utilizado para trabalhar no banco de dados e também na interface do usuário.

#2 Muitas opções de bibliotecas e frameworks que vão maximizar sua produtividade.

#3 Fácil compreensão das funções da linguagem

#4 Linguagem muito popular e adotada pelas principais empresas de tecnologia do mundo.

#5 Linguagem sempre atualizada com novos frameworks, fazendo com que sempre consiga suprir alguma necessidade.

#6 Existe muito conteúdo na internet relacionado à linguagem, ajudando a aprender ainda mais.

#7 A linguagem é multiplataforma

#8 Pelas múltiplas possibilidades que a linguagem oferece e muito trablhada com CSS e HTML, cria-se muitas formas de resolver problemas

#9 Ótima linguagem para praticar programação e se desenvolver na área
